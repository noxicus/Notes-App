Andorid App for managing every day notes. It enables adding, editing, deleting of notes.
Room database is used for storing data and implements MVVM arhitecture.
- Room Database (SQLite)
- Data Access Objects (DAO)
- View Model
- Live Data
- Animations
- Swipe to delete
